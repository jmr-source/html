安装与卸载

安装

下载地址

Jmr下载地址：http://www.jmr-source.com/downloads/jmr_1.0.0/site.xml

适用版本

  • Eclipse 4.2 ~ 4.6，包含各种版本都可以直接安装，如：
    • Java版
    • Java EE版
    • Android版
    • PHP版
    • C/C++
    • JavaScript
    • 其它更多版本...
  • Eclipse 3.5 ~ 3.7，Java EE版本可直接安装。
  • MyEclipse 9 10 2014 2015 2016。

目录:

  • Eclipse安装
  • MyEclipse安装

Eclipse安装

在Eclipse菜单栏中，点击Help -> Install New Software -> Add

勾选Basic、Jet和Jmr，点击Next直到下载安装完毕并重启Eclipse

一路Next直到安装结束，如果安装比较慢或者卡住无法安装，请把Contact all update...勾除，并重新安装。

安装结束后提示重启，重启后会提示开始初始化，点击是，开始初始化Jmr组件

初始化组件后，再次重启Eclipse

安装成功，可以看到工具栏中Jmr的图标

卸载

点击工具栏中的卸载

点击是，开始卸载

要彻底删除Jmr插件，请点击是

点击确认

选中Jet和Jmr组件，点击Uninstall完成最后卸载，并重启Eclipse即可

MyEclipse安装

在菜单栏中选择Window（窗口）-->Preferences（首选项）-->General（常规）-->Capabilities（功能），勾选Classic Update，点击Apply（应用），重启MyEclipse。

后续的安装同Eclipse，不过有几点操作需要注意。

  • 需要勾掉Contact all update...的选项。

  • MyEclipse 2014 2015 2016，在安装到此页面时，选择Update my installation the same...

  • 安装完成后，如果还不能使用，请重新启动MyEclipse即可。
