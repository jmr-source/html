MID() 函数

MID() 函数用于从文本字段中提取字符。

SQL MID() 语法

SELECT MID(column_name,start[,length]) FROM table_name;

 

参数

描述

column_name

必需。要提取字符的字段。

start

必需。规定开始位置（起始值是 1）。

length

可选。要返回的字符数。如果省略，则 MID() 函数返回剩余文本。

SQL MID() 实例

下面的 SQL 语句从 "Websites" 表的 "name" 列中提取前 4 个字符：

实例

SELECT MID(name,1,4) AS ShortTitle
FROM Websites;
